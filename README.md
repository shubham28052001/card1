## Overview  
This project presents a responsive **cards grid interface** where items (menu entries) are displayed as cards in a flexible grid layout. It features category-based filtering and functions well across mobile, tablet and desktop screen sizes.

## Features  
- Responsive grid layout using CSS Grid (`repeat(auto-fit, minmax(...))`).  
- Category filter buttons to dynamically show/hide card items.  
- Clean card design with image, title, category, price & description.  
- Mobile-first CSS with media queries for improved usability on different viewports.  
- Lightweight implementation (vanilla JS, no frameworks).
